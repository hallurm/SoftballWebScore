# SoftballWebScore

## Beskrivelse
SoftballWebScore er et WordPress-baseret system til live score-opdatering for softballkampe. Det er specifikt udviklet til Egedal Softball Club's hjemmeside (https://egedalsoftball.dk).

## Funktioner
- Live score-opdatering
- Custom Post Type for kampe
- REST API til score-opdateringer
- AJAX-baseret frontend for realtidsvisning af scores

## Teknologier
- WordPress
- Advanced Custom Fields (ACF) Pro
- Quform plugin
- X Pro tema fra theme.co

## Installation
1. Klon dette repository til din WordPress-installation
2. Aktiver de nødvendige plugins: ACF Pro og Quform
3. Aktiver X Pro temaet
4. Konfigurer ACF-felter som beskrevet i `beskrivelse.md`
5. Opret den nødvendige Quform som beskrevet i `beskrivelse.md`

## Brug
Se `beskrivelse.md` for detaljerede instruktioner om, hvordan systemet skal konfigureres og bruges.
